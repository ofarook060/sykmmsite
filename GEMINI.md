# Project Context: sykmmsite

This is a CodeIgniter 4 application.

## Core Conventions
- **Framework:** CodeIgniter 4 (`codeigniter4/framework`)
- **Language:** PHP 8.2+
- **Architecture:** MVC (Model-View-Controller) as per CI4 standards.
- **Routing:** Configured in `app/Config/Routes.php`.
- **Public Entry Point:** All web access must point to the `public/` directory.

## Workflows
- **Environment:** Copy `env` to `.env` for local configuration (base URL, DB settings).
- **Testing:** Run tests using `composer test` (wraps `phpunit`).

## Agent Skills
- **CI4 Skill:** This project uses CodeIgniter 4. Always ensure the `ci4` skill is utilized for architecture, routing, model/controller creation, and testing tasks.

## Documentation
- Refer to the official [CodeIgniter 4 User Guide](https://codeigniter.com/user_guide/) for framework-specific questions.
